# 🧠 Assistente Financeiro — Prompts

Este arquivo contém os prompts principais para um assistente financeiro simples e prático.

---

# 🧩 1. PROMPT BASE (COMO A IA DEVE SE COMPORTAR)

Você é um assistente financeiro simples.

Seu trabalho é ajudar pessoas a organizarem melhor o dinheiro.

Regras:
- Use linguagem simples
- Explique de forma clara
- Evite termos difíceis
- Dê respostas práticas
- Use exemplos do dia a dia
- Se faltar informação, faça perguntas simples

---

# 📊 2. ENTENDER MINHA SITUAÇÃO

Me ajude a entender minha situação financeira.

Explique:
- se estou bem ou mal financeiramente
- onde estou gastando demais
- o que devo melhorar primeiro

---

# 💰 3. ORGANIZAR DINHEIRO

Me ajude a organizar meu dinheiro.

Divida meu salário em partes simples:
- contas essenciais
- gastos pessoais
- economia ou investimentos

---

# 💸 4. ECONOMIZAR DINHEIRO

Me ajude a gastar menos dinheiro.

Mostre:
- o que posso cortar
- o que devo repensar
- dicas simples para economizar

---

# 💳 5. DÍVIDAS

Me ajude a sair das dívidas.

Explique:
- o que pagar primeiro
- como evitar novas dívidas
- como organizar de forma simples

---

# 📈 6. INVESTIMENTOS

Me explique como começar a investir com pouco dinheiro.

Explique:
- por onde começar
- o que é seguro
- o que evitar

---

# 💰 7. CARTEIRA SIMPLES

Me ajude a montar uma carteira simples de investimentos.

Inclua:
- reserva de emergência
- opções seguras

---

# ⚖️ 8. DECISÕES FINANCEIRAS

Me ajude a decidir se devo comprar algo.

Considere:
- se eu preciso disso
- se cabe no orçamento
- se vale a pena

---

# ⚖️ 9. COMPARAR OPÇÕES

Compare:
[OPÇÃO A] vs [OPÇÃO B]

Explique:
- qual é melhor financeiramente
- qual é mais seguro
- qual faz mais sentido no longo prazo

---

# 📚 10. APRENDER SOBRE DINHEIRO

Me ensine sobre dinheiro de forma simples.

Crie um passo a passo:
- organizar dinheiro
- evitar dívidas
- começar a investir

---

# 🧠 11. MENTORIA FINANCEIRA

Atue como meu mentor financeiro.

Sempre:
1. Explique de forma simples
2. Diga o que fazer
3. Dê um passo a passo
4. Mostre erros comuns
