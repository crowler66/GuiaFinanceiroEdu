# 📚 Guia do Assistente Financeiro

Este guia explica como usar o Assistente Financeiro de forma simples.

---

# 🎯 O que é este projeto

Este é um assistente financeiro simples que ajuda pessoas a:

- Entender o próprio dinheiro
- Organizar gastos
- Sair de dívidas
- Economizar mais
- Começar a investir

---

# ⚙️ Como funciona

O sistema usa uma base de textos e prompts que ajudam a IA a responder perguntas financeiras de forma simples e prática.

Ele funciona como um guia de conversa sobre dinheiro.

---

# 💬 Como usar

Você pode fazer perguntas simples como:

- Como organizar meu dinheiro?
- Como sair das dívidas?
- Como gastar menos?
- Vale a pena comprar isso?
- Como começar a investir?

---

# 🧠 Como o assistente responde

Ele sempre deve:

- Usar linguagem simples
- Dar respostas diretas
- Explicar com exemplos do dia a dia
- Mostrar o que fazer na prática
- Evitar termos difíceis

---

# 📊 O que ele pode te ajudar a fazer

- Criar um orçamento simples
- Entender sua situação financeira
- Cortar gastos desnecessários
- Organizar dívidas
- Começar a investir com pouco dinheiro

---

# 🚀 Objetivo

Ajudar qualquer pessoa a melhorar sua vida financeira de forma simples e clara.
