# 🧠 Assistente Financeiro Inteligente

Este é um projeto de assistente financeiro simples e educativo, criado para ajudar pessoas a organizarem melhor o dinheiro e tomarem decisões financeiras mais conscientes.

---

# 🎯 Objetivo do projeto

Este projeto foi criado para ajudar qualquer pessoa a:

- Entender sua situação financeira
- Organizar o dinheiro do mês
- Controlar gastos
- Sair de dívidas
- Criar o hábito de economizar
- Começar a investir de forma simples
- Tomar melhores decisões financeiras no dia a dia

---

# ⚙️ Como funciona

O projeto funciona como uma base de conhecimento estruturada.

Ele pode ser usado em ferramentas como NotebookLM ou outras IAs que trabalham com documentos.

A IA usa esses arquivos como referência para responder perguntas financeiras de forma simples, direta e prática.
